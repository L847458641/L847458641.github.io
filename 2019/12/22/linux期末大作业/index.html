linux期末大作业

移动互联187 李晓伟 181203250

前言：

搭建项目：使用hexo+github搭建个人博客

域名绑定：www.lixiaowei.club

下载安装Node.js

选择你的安装路径，然后一直点Next，安装。

打开cmd，输入node -v和npm -v，出现版本号，则安装成功

下载安装Git

参考https://www.jianshu.com/p/414ccd423efc

cmd 输入 git –version出现版本号则安装成功

注册github账号

在右边点击+号，新将一个仓库，仓库命名：你的github名字.github.io

选择public，勾选Initialize this repository with a README

点绿色按钮创建仓库

进入仓库，点击settings，向下拉到最后有个GitHub Pages，点击Choose a theme选择一个主题。选择好主题后，github pages 会有一个链接，点击可以看到你自己的博客网页

安装Hexo

在d盘或者其他地方新建一个文件夹blog

进入blog文件，右击打开Git Bash Here

输入npm i hexo-cli -g安装Hexo

安装完后输入hexo -v验证是否安装成功

输入hexo init初始化文件夹，接着输入npm install安装必备的组件

输入hexo g生成静态网页，然后输入hexo s打开本地服务器

浏览器打开http://localhost:4000/，就可以看到我们的博客

按ctrl+c关闭本地服务器

连接Github与本地

Git Bash Here输入

git config –global user.name “githubname”

git config –global user.email “邮箱”

生成密钥SSH key

在C:\Users\g.ssh里会生成两个文件 id_rsa和id_rsa.pub

记事本看到id_rsa.pub复制里面的密钥到github创建ssh的框里面

打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便。

输入ssh -T git@github.com，如果如下图所示，出现你的用户名，那就成功了

Blog文件打开博客的配置文件：_config.yml

修改最后一行的配置：

Deploy:

type: git

repository :https://github.com/githubname/githubname.github.io.git

branch :master

写文章发布到博客

Git Bash Here 安装一个扩展：npm i hexo-deployer-git

再输入hexo new post "article title"，新建一篇文章。

在blog/source_posts,多了个.md文件,

输入hexo g 生成静态网页

输入hexo d上传

这时打开你的github.io主页就能看到发布的文章啦

绑定域名：不用输入githubname.github.io,直接输入域名就可以访问博客

具体请看下一篇博客文章
